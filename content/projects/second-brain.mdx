This is a full-stack web application built to help users save and organize
useful online content in one place. It supports YouTube videos, Twitter/X posts,
and websites, allowing users to revisit important resources whenever needed.

The application uses JWT authentication for secure access and provides a
shareable link feature to publicly share saved content collections.

## Live Demo

Visit the application here:

[Second Brain Live Demo](https://second-brain-rust-seven.vercel.app/)

## Features

- **Authentication**: Secure sign up and sign in using JWT
- **Content Management**: Save YouTube videos, Twitter/X posts, and websites
- **Tagging System**: Organize saved content using custom tags
- **Filtering**: Filter content by type and tags
- **Shareable Brain**: Generate public links for sharing saved content
- **Responsive Design**: Works across different screen sizes
- **Dashboard UI**: Clean and intuitive interface

## Technologies

- **React (TypeScript)**: Frontend framework
- **Tailwind CSS**: Styling and responsive design
- **Node.js & Express**: Backend API and server logic
- **MongoDB & Mongoose**: Database for storing content
- **JWT Authentication**: Secure route protection
- **Vercel**: Frontend deployment
- **Render**: Backend hosting

## Getting Started

Clone the repository:

```bash
git clone git@github.com:Pankajkantghz/second-brain.git
cd second-brain
```

Install dependencies:

```bash
npm install
```

Start frontend:

```bash
cd frontend
npm run dev
```

Start backend:

```bash
cd backend
npm run dev
```

The frontend runs at:

```text
http://localhost:5173
```

Backend API runs at:

```text
http://localhost:3000
```

## Deployment

The frontend is deployed using Vercel and the backend is hosted on Render.

The application includes:

- JWT authentication
- Protected routes
- Secure CORS configuration
- Rate limiting
- Secure API practices

## Conclusion

Second Brain is designed as a personal knowledge hub where users can save,
organize, and revisit useful online resources. With tagging, filtering, and
public sharing features, it provides a simple way to build a personal digital
second brain.
